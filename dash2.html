Принцип отображения сотрудников в блоках:

  • В блоке “Сегодня” отображаются ДР сотрудников текущего дня до 00.00
  • В блоке “На этой неделе” отображаются ДР с завтрашнего дня 00.01 до воскресенья текущей недели 00.00

Дополнительные требования по отображению блоков:

  • Если есть ДР сегодня, отображать блок “сегодня”, если нет - не отображать блок
  • Если есть ДР на этой неделе, отображать блок "на этой неделе", если нет - не отображать блок
  • Если есть ДР позже воскресенья этой недели, отображать блок “ближайшие”, если нет - не отображать блок

 

Дополнительные требования по фильтрации сотрудников в блоках:

  • Фильтрация в блоке "Сегодня" по ФИО, как и везде в системе
  • Фильтрация в блоке “На этой неделе” и в блоке “Ближайшие” по дате ДР (ближайшая дата - сотрудник выше), если дата одинаковая - по ФИО
